moduli.FunzioniMenu.creaVocimenu(tup)[sorgente]

Funzione che genera le voci di menù originate dalla tupla di stringhe in ingresso la sequenza contiene tutte le voci diverse da «<Inattiva>»

moduli.FunzioniMenu.menu(tup)[sorgente]

moduli.ChiediDati.chiediIntero(prompt, da=-1000000000, a=1000000000)[sorgente]

Chiede un intero in ingresso, stampando una frase di richiesta il numero deve essere strettamente compreso fra «da» e «a», scartando eventuali input errati

moduli.ChiediDati.chiediStringa(prompt)[sorgente]

Documentazione Funzioni Orario

moduli.FunzioniOrario.conta_ore(riga_file)[sorgente]

Conta il numero di ore

Primo parametro - accetta in ingresso una riga del file già separata, depurata e inserita in una lista

moduli.FunzioniOrario.docenti_classe(file_orario, classe, file_output)[sorgente]

Scrive in un file l’elenco dei docenti di una data classe

Primo parametro - accetta in ingresso il percorso del file da analizzare
Secondo parametro - accetta in ingresso una classe dell’orario
Terzo parametro - accetta in ingresso il percorso del file in cui scrivere l’elaborazione

moduli.FunzioniOrario.orario_docente(file_orario, docente, file_output)[sorgente]

Scrive in un file l’orario di un determinato docente e numero totale delle ore del docente

Primo parametro - accetta in ingresso il percorso del file da analizzare
Secondo parametro - accetta in ingresso un docente dell’orario
Terzo parametro - accetta in ingresso il percorso del file in cui scrivere l’elaborazion

moduli.FunzioniOrario.ore_diposizione(file_orario, docente, file_output)[sorgente]

Scrive in un file le ore a disposizione di un determinato docente

Primo parametro - accetta in ingresso il percorso del file da analizzare
Secondo parametro - accetta in ingresso un docente dell’orario
Terzo parametro - accetta in ingresso il percorso del file in cui scrivere l’elaborazione

moduli.FunzioniOrario.ore_disponibile(file_orario, ora, giorno, file_output)[sorgente]

Scrive in un file l’elenco dei docenti che hanno lezione a una determinata ora di un determinato giorno della settimana

Primo parametro - accetta in ingresso il percorso del file da analizzare
Secondo parametro - accetta in ingresso un’ora
Terzo parametro - accetta in ingresso un giorno
Quarto parametro - accetta in ingresso il percorso del file in cui scrivere l’elaborazione
